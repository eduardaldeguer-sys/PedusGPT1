PedusGPT Beta 1
Un transformer decoder-only construido desde cero en PyTorch
🔤
Tokenizador BPE
Byte Pair Encoding implementado desde cero. 8,000 tokens de vocabulario entrenados sobre el corpus.
🧠
Transformer GPT
6 bloques, 8 cabezas de atención causal, dimensión 512. ~25M parámetros totales.
⚡
Mixed Precision
BFloat16 + gradient accumulation. Optimizado para RTX 5060 Ti con 16 GB VRAM.
Arquitectura
El modelo sigue la arquitectura GPT-2 con Pre-LayerNorm (más estable que Post-LN). Cada bloque tiene atención multi-cabeza causal con máscara triangular, seguida de una red feed-forward de 2 capas con activación GELU. Los embeddings de token y posición se suman antes del primer bloque. El lm_head comparte pesos con el embedding de tokens (weight tying).

Datos
Por defecto se usa WikiText-2 (~2M tokens) para entrenar rápido y verificar que todo funciona. Para un modelo más capaz, cambia a WikiText-103 (~100M tokens) editando config.py.
